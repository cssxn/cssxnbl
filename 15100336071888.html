编写驱动前需要的准备

2017/11/7

最近学习了windows内核编程，在开始编写驱动程序之前，需要修改一下系统的注册表，否则WinDBG不能显示驱动程序打印的调试信息

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Debug Print Filter]
 
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Debug Print Filter]
"DEFAULT"=dword:0000000f

在windows注册表找到上面的Debug Print Filter注册表项，如果没有则需要手动创建一个Key

创建完成后，右键菜单刚才创建的Debug Print Filter，选择New->DWORD(32-bit) Value

然后改名为default，然后双击修改它

将default的值0改为f即可

修改完成后，需要重启虚拟机的windows系统才能生效！
