Windows开启防火墙后ping不通

2018/2/9

VMWare虚拟机中安装Win7后，主机ping不通虚拟机，后来发现关闭防火墙后可以ping通，但是又不想关闭防火墙

记得以前学网络协议的时候ping命令的实现是发送icmp包，原来开启防火墙默认关闭了icmp回显示，导致ping不通的效果，要解决这个问题很简单，开启icmp回显就行了！

以管理员权限启动CMD，然后输入下面命令

开启icmp回显

netsh firewall set icmpsetting 8

关闭icmp回显

netsh firewall set icmpsetting 8 disable
